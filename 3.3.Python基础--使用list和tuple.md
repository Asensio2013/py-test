### list
* Python内置的一种数据类型是列表: list
    * list 是一种有序的集合，可以随时添加和删除其中的元素。


* 声明:
    * 用中括号:
    * `l = ['lin', 'wang'];`


* 长度:
    * `len()` 函数可以获得 list 元素的个数


* 访问:
    * 用索引来访问list中每一个位置的元素, 索引是从 0 开始

    * 访问索引越界报 `IndexError` 错误

    * `l[0]` || `l[1]` || `l[-1]`


* 追加:
    * `append()` 函数

    * `l.append('zhang')`


* 插入:
    * `insert()` 函数

    * `l.insert(1, 'que')`


* 删除:
    * `pop()` 函数

    * `l.pop(1)`

* 替换:
    * 也就是直接赋值

    * `l[1] = 'xxx'`


* 二维数组:
    * list元素也可以是另一个list

    * `s = ['python', 'java', ['asp', 'php'], 'scheme']`
